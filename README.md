# fj

[![Build Status](https://travis-ci.org/ymyzk/fj.svg?branch=master)](https://travis-ci.org/ymyzk/fj)

## Requirements
* Menhir
* OCaml
* ocamlfind
* OMake
* OUnit

## Compile & Run
* `omake`
* `./src/main <source.java>`

## Test
* `omake test`
